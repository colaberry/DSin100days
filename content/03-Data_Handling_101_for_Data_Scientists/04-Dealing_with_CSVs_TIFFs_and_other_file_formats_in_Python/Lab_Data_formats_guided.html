Reading the MNIST Dataset¶

In this part you will read the MNIST dataset. You will need to know how to read an idx file. You can find more information about this here. The format of the file is such that:

For the training images:
1) The first four bytes of the file correspond to the magic number. This is an identifer for the file
2) The next four bytes correspond to the number of images in the training set
3) The next four bytes correspond to the x-dimension of the image
4) The next four bytes correspond to the y-dimension of the image
5) The rest of the bits in the file correspond the image data.

You will need to reshape the data you get from point 5) onwards since the data is suppose to be image data. Keep in mind that the MNIST dataset contains greyscale images where the pixel values go between 0 and 255.
